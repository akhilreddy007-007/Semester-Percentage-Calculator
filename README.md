# Semester-Percentage-Calculator

## 📌 Overview
A Java console application to calculate semester-wise percentages using grades entered by the user.

---

## 🚀 Features
- Supports 8 semesters
- Calculates semester percentage
- Displays combined percentage
- Uses inheritance concept
- Console-based application
- Beginner-friendly Java project

---

## 🛠 Technologies Used
- Java
- OOP Concepts
- Scanner Class
- Arrays
- Inheritance

## ▶️ How to Run

- 1. Compile files
- javac *.java

- 2. Run program
- java main


## How It Works
- User enters semester number
- User enters grades
- Program calculates marks
- Percentage is displayed
